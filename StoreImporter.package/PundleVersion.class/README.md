PundleVersion is the abstraction for a single bundle or package version in Store.

Instance Variables:
	logger			<Settlogger>		writes messages to the log file
	primaryKey	<Integer>			primary key in the bundle or package table
	pundleName	<String>			name of bundle or package
	timestamp		<DateAndTime>	when the code was published to Store
	username		<String>			userid of the develper who published
	version		<String>			human meaningful label for the version
