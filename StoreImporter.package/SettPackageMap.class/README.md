SettPackageMap holds the class definitions and methods for a given namespace-specific package, along with the prefix to be applied to the package name when writing the tonel file.

Instance Variables:
	classes		<Collection of: SettClassDefinition>	class definitons along w/methods
	extensions	<Collection of: SettClassDefinition>	class definitions w/usually just methods
	prefix			<String>										prefix added to package name when writing the tonel file
